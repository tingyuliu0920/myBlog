#### Create a Hexo site

```bash
hexo init [folder]
```

#### Generate Static Files

```bash
hexo generate
```

#### Serve the Site Locally

```bash
hexo s
```

#### Create a New Post

```bash
hexo new [title]
```

#### Create a New Page

```bash
hexo new page [pageName]
```

#### Create a New Draft

```bash
hexo new draft [draftTitle]
```

#### Publish a New Draft

```bash
hexo publish [draftTitle]
```

#### Preview drafts on the server

```bash
hexo s --draft
```
